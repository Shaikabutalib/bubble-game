Dynamic Bubble Generator with Interactive Gameplay

This is an interactive web application that generates bubbles with random numbers inside them. The number of bubbles adjusts dynamically to fit the size of the container. Additionally, the application includes gameplay elements where users need to click on bubbles matching a randomly generated target number within a given time limit to score points.

Introduction
This project builds upon the dynamic bubble generator concept and incorporates interactive gameplay features. Users are presented with a grid of bubbles containing random numbers. They must click on bubbles that match a randomly generated target number within a specified time limit to score points. The game ends when the timer runs out.

Features
Dynamically generates bubbles to fit within the container.
Interactive gameplay where users must click on bubbles matching a target number to score points.
Timer countdown to create urgency and challenge.
Responsive design ensures optimal display on different devices.
Usage

To use this project:
Clone the repository to your local machine.
Open the index.html file in a web browser.
Click on bubbles matching the target number within the specified time limit to score points.

How It Works
The application utilizes JavaScript to dynamically generate bubbles based on the dimensions of the container. It also includes gameplay logic to generate a target number, start a timer, and handle user interactions. Bubbles matching the target number increment the score, and the game ends when the timer runs out.
